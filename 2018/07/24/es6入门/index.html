es6入门

发表于 2018-07-24 | 分类于 学习笔记

一：命令

  1. let命令是es6新增的声明变量的命令，用户类似于js的var，但是let声明的变量只在所在的块内生效。
  2. const命令声明了一个不允许改变的常量，即一旦声明，该常量的值就不能再改变。并且在使用过程中一旦使用const声明了变量，就必须立刻给其赋值，不能留到以后再赋值，否则会报错。

二：解构

  1. es6允许对数组按照对应位置进行赋值，但是“=”两边的模式必须是匹配的

  2. 对对象进行解构赋值，与数组不同的是，对对象进行解构时，变量必须与属性同名，才能取到正确的值。

  1. 当解构赋值时“=”两边是数值或者布尔值，会先将其转换成对象，然后再进行解构。

  2. 解构常用的的用途：

    1. 交换变量的赋值
    2. 从函数返回多个值（一个函数只能返回一个值，当需要返回多个值的时候，将其放在数组或者对象里，然后再通过解构将其取出来）
    3. 对函数的参数进行定义
    4. 提取json对象里的数据。

三：字符串的扩展

  1. 在JS中只提供了indexof方法用来确定一个字符串是否包含在另一个字符串中，es6还提供了三种新的方法：

    includes() 返回布尔值 表示是否找到了参数字符串

    startsWith() 返回布尔值 表示参数字符串是否在原字符串的头部

    endsWith() 返回布尔值 表示参数字符串是否在原字符串的尾部

  2. repeat（）方法将会根据参数返回一个新的字符串，参数表示将原字符串重复几次。（参数只能大于0，如果是小数将会取整）

  3. 在es6中引入了模板字符串，模板字符串是增强版的字符串，用反引号（ ` ）标识。

注意：

  1. 如果在模板字符串中需要使用反引号，则需要反斜杠()转义，并且在模板字符串中使用的空格和换行将会被输出。
  2. 在模板字符串中嵌入变量时，需要将变量名写在${}中，在{}中可以写入任何的JS表达式，甚至可以进行运算、引用对象属性等。
  3. 在模板字符串中也可以调用函数，但是如果变量没有声明，将会报错。
  4. 并且在变量中可以进行多层嵌套，比如说在其中再嵌套一个模板字符串。

四：数值的扩展

  1. es6规定：二进制要用前缀0b表示；八进制要用前缀0o表示。如果需要将带有前缀的字符串转为十进制需要使用Number方法。
  2. es6新提供了两个方法：Number.isFinite()和Number.isNaN()。前者用来检查一个数值是否为有限数值，如果参数不是数值，那么将一律返回false；后者用来判断参数值是否为NaN,同理如果参数不是NaN，一律返回false。
  3. es6使用Number.isInteger()来判断一个数值是否为整数，其中整数和浮点数会被视为同一个值，如果方法的参数不是数值，将会返回false。（如果对于数值的精度要求很高，就不要使用这个方法）
  4. 有关Math对象的扩展：
    • Math.trunc()方法用于去除一个数的小数部分，返回整数部分
    • Math.sign()方法用来判断一个数到底是正数、负数、还是零。对于非数值，会先将其转换为数值。

五：函数的扩展

  • 函数参数的默认值

  1. 基本用法：在es6中，可以直接为函数的参数设置默认值，并且直接写在参数定义的后面。（因为参数变量是默认声明的，所以不能再使用const或者let声明，并且函数的参数不能有同名的）。注意：参数默认值是赋值而不是传值即每次调用都会重新赋值。

  2. 与解构赋值默认值结合使用。

  3. 函数的length属性将返回没有指定默认值的参数个数，如果都指定了默认值，那么length属性就会失真。（如果指定了默认值的参数不是尾参数，那么length属性也不会计入后面的参数。）

  4. 当设置了参数的默认值之后，参数会形成一个单独的作用域，在作用域之外访问，是访问不到的。（如果在其单独的作用域没有定义变量，将会指向外层的全局变量）

    es6规定：只要函数使用了默认值、解构赋值、或者扩展运算符，那么该函数内部就不能显示设定为严格模式。
